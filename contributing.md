## convention

### prettier 
- single, double, quote? => single: 세현, 현준 / double: 민경
- tab width => 2: 세현, 현준, 민경 / 4: 민경
- semi: true: 세현, 현준, 민경 /
- printWidth: pass

### 폴더명
- snake case -> 세현

### typescript
- I or Props -> Props: 현준, 민경
- props type -> 컴포넌트 상단: 현준, / 무관: 민경 
- type + interface naming: pascal: 현준, cammel: 민경
